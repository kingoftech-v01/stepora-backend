**Date:** 2026-03-26
**Repos audited:** `/root/stepora` (backend), `/root/stepora-frontend` (frontend), `/root/stepora-mobile` (mobile)

**Method:** Code review against vulnerability checklists. REPORT ONLY -- no fixes applied.
